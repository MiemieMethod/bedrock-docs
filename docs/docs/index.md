# 文档

这里为Minecraft基岩版开发者提供了一系列的文档。文档围绕各类概念与主体展开，以较为正式的语气和说明性的体裁呈现。本站正在持续补充更多条目，以便覆盖开发Minecraft基岩版内容时需要理解的主要概念。

本文档主要回答“是什么”和“为什么”。需要了解实际制作步骤时，请查看教程；需要查阅字段、参数或API成员时，请查看参考。

## 文档板块

文档板块按内容面向分为若干部分：

- **通识**：介绍版本、数据驱动、游戏对象、命令、诊断工具和界面系统等基础概念。
- **附加包**：介绍资源包、行为包、清单文件、包加载机制以及各类资源、文本和数据定义。
- **服务器**：介绍基岩版专用服务器、网络协议、插件和插件加载器。
- **客户端模组**：介绍InnerCore旧客户端模组生态及其关键构成。
- **存档**：介绍世界存档及其持久化数据结构。
