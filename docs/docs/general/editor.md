# 编辑器<!-- md:flag vanilla -->

**编辑器（Editor）**是Minecraft基岩版国际版中提供的一种专用世界编辑工具模式。编辑器为创作者提供了一套图形化的创作界面，用于高效地构建和编辑世界内容。

## 概述

Minecraft编辑器是一个运行在基岩版引擎上的独立模式。与普通的游戏模式不同，编辑器模式提供了一系列专业的创作工具，包括区域选择、方块填充、刷子工具、结构复制粘贴等功能，旨在简化大型建筑和地图的创作流程。

编辑器目前处于早期开发阶段，功能仍在持续扩展中。编辑器仅在Windows平台可用。

## 主要功能

### 工具面板

编辑器提供了一组工具面板，用于执行不同的编辑操作：

- **选择工具**：选取一个三维区域用于后续操作。
- **刷子工具**：使用可配置形状和大小的刷子在世界中快速绘制方块。
- **填充工具**：将选定区域内的方块替换为指定方块。
- **线条工具**：在两个点之间绘制直线。

### 视角控制

编辑器提供自由的飞行相机视角，创作者可以在不受物理限制的情况下自由浏览和编辑世界。相机的移动速度可以调节，支持快速移动到世界的任意位置。

### 撤销与重做

编辑器内置了多步撤销和重做功能，允许创作者安全地尝试和回退修改操作。

### 结构操作

编辑器支持对选定区域进行复制、剪切、粘贴、旋转和镜像等操作，方便快速构建对称结构或重复模式。

## 编辑器扩展

编辑器支持通过脚本API创建自定义扩展，扩展编辑器的功能。编辑器扩展可以添加自定义工具、面板和操作，使创作者能够根据需要定制创作流程。编辑器扩展使用`@minecraft/server-editor`脚本模块。

## 编辑器与存档

编辑器模式创建的世界可以导出为普通的游戏存档，也可以将普通存档导入编辑器进行编辑。编辑器模式的存档与普通存档在数据格式上兼容。
