# 编辑器<!-- md:flag vanilla -->

**编辑器（Editor）**是Minecraft基岩版国际版中提供的一种专用世界编辑工具模式。编辑器为创作者提供图形化的创作界面，用于高效构建、编辑、测试和导出世界内容。

## 概述

Minecraft编辑器是运行在基岩版引擎上的独立启动模式。与普通游玩流程不同，编辑器在游戏画面上叠加专用窗口、菜单、工具栏和操作栏，使创作者能够在世界内部直接进行多方块编辑。编辑器面向键盘和鼠标使用，目前仅在Windows电脑上可用，不适用于移动端或主机。

编辑器可从Minecraft启动器的**Creator Tools**页进入，也可以通过`minecraft://creator/?Editor=true`或`minecraft-preview://creator/?Editor=true`协议快捷方式启动。Microsoft Learn当前说明，零售版和预览版基岩版均可以进入编辑器；预览版仍可能包含更新但不稳定的编辑器功能。

## 项目与世界

编辑器使用**项目（Project）**作为编辑工作空间。项目保存编辑状态、项目设置、导出设置和用于编辑的世界数据。项目可以导出为可游玩的`.mcworld`世界，也可以在测试世界流程中临时以普通玩法方式运行。项目不应简单等同于最终发布的世界文件；它更接近用于持续编辑和导出的源数据。

创建项目时，创作者可以设置项目名称、导出相关设置和通常的世界选项。项目启动后，编辑器会显示欢迎或快速开始窗口，用于说明移动方式、工具入口和常用链接。

## 界面

编辑器界面由多个可移动、可折叠、可关闭的窗口组成。常见区域包括：

- **菜单栏**：提供文件、视图、帮助、界面设置等入口。
- **工具栏**：列出选择、刷子、填充、线条、铅笔、挤出等编辑工具。
- **操作栏**：放置可快速执行的常用动作。
- **方块快捷栏**：选择当前用于编辑的方块。
- **工具窗口**：显示当前工具的形状、尺寸、模式和其他参数。
- **指南针**：显示当前朝向，并可用于快速对齐视角。

编辑器还提供界面缩放、字体大小和主题等UI设置。这些设置只影响编辑器界面显示，不改变项目内容。

## 主要功能

### 工具系统

编辑器提供一组用于世界编辑的工具。选择工具用于选取三维区域；刷子工具用于按形状快速绘制方块；填充工具用于替换或填充区域；线条、铅笔、挤出、缩放、尺规等工具用于处理不同类型的构建任务。不同工具通常具有各自的参数窗口和快捷键。

### 移动与视角

编辑器提供自由飞行相机。工具模式下通常需要按住鼠标右键再使用++w++、++a++、++s++、++d++、++space++和++shift++移动。视图设置中可以调整飞行速度。对准远处方块按++g++可以使用视线传送功能移动到目标位置。

### 撤销、重做与结构操作

编辑器内置撤销和重做功能，允许创作者回退或恢复编辑操作。编辑器还支持对选定区域执行复制、剪切、粘贴、旋转、镜像和预览粘贴等结构化操作，适合处理大型建筑、重复构件和对称结构。

### 世界选项

世界选项提供对编辑过程中常见世界状态的快速控制，包括暂停实体模拟、切换维度、调整时间、修改天气、打开导航面板以及进入测试世界。测试世界会以可游玩方式运行当前项目；测试期间所做的改动不会保存回项目，退出测试后可返回编辑器继续编辑。

## 编辑器扩展

编辑器支持通过脚本API创建扩展。编辑器扩展可以添加自定义工具、面板、操作和工作流，用于扩展默认编辑能力。编辑器扩展使用`@minecraft/server-editor`脚本模块，并依赖编辑器环境提供的接口。

## 与BDS

Windows版[基岩版专用服务器](../server/bds.md)可以承载编辑器项目。以命令行参数`Editor=true`启动BDS时，服务器可以从空世界创建编辑器项目；也可以将已有编辑器项目放入服务器{{file|worlds}}目录，再通过{{file|server.properties|txt}}指定`level-name`。在这种状态下，客户端连接服务器会进入编辑器会话，且客户端版本必须与服务器版本匹配。

## 与存档

编辑器项目可以导出为普通世界，也可以将普通世界导入编辑器进行编辑。编辑器项目与普通世界共享基岩版世界数据基础，但项目还包含编辑器工作流所需的附加状态和导出信息。因此，用于正式发布、备份和跨设备分发时，通常应导出为`.mcworld`或复制完整项目目录，并避免只保存部分世界文件。
