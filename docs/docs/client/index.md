# 客户端模组

**客户端模组（Client Mod）**是运行在Minecraft基岩版客户端侧、直接扩展或修改客户端行为的一类模组形态。与附加包和服务端插件相比，客户端模组通常依赖特定加载器、特定系统环境和特定游戏版本，兼容性边界更严格。

## 概述

在基岩版历史生态中，客户端模组主要以Android平台的InnerCore体系为代表。该体系通常由启动器、模组包和引擎接口共同构成，允许脚本、Java与原生代码协同工作。

## 适用范围

本分区记录旧客户端模组生态中的核心术语和结构性概念，主要用于：

- 维护历史项目；
- 理解早期社区技术路线；
- 为迁移到附加包、脚本API或服务端插件提供背景。
