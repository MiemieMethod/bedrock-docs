# Horizon启动器

**Horizon启动器（Horizon Launcher）**是InnerCore生态中用于安装、管理和启动模组包的客户端程序。该启动器在旧客户端模组流程中承担入口角色，负责将模组包与游戏运行环境连接起来。

## 功能

Horizon启动器通常提供以下能力：

- 下载与安装不同模组包版本；
- 管理模组与模组包列表；
- 提供日志与故障信息入口；
- 启动对应包内的游戏运行环境。

## 兼容性因素

Horizon启动器的可用性受Android系统版本、存储权限模型和包版本共同影响。不同Android版本下，包目录访问方式与文件管理流程可能存在显著差异。

## 与InnerCore的关系

Horizon启动器并不等同于InnerCore。前者是启动和管理平台，后者是面向模组的运行框架；二者协同构成旧客户端模组开发与分发链路的一部分。
