# ModPE

**ModPE（ModPE Script）**是早期Android版Minecraft基岩版社区中用于客户端脚本扩展的一类接口体系，主要与BlockLauncher等启动器生态相关。其核心价值是以较低门槛在客户端侧注入脚本逻辑，用于修改交互、界面与部分玩法行为。

## 定位

ModPE属于历史客户端模组路线，不属于官方附加包体系，也不属于现代服务端插件体系。其可用性通常受启动器版本、Android系统版本与目标游戏版本共同约束。

## 生态特征

ModPE生态通常具有以下特征：

- 以脚本扩展为主，强调快速修改与轻量分发；
- 运行环境依赖特定启动器实现；
- 与InnerCore、LeviLamina客户端模组在接口模型和运行机制上互不等价。

## 现状

ModPE在基岩版早期移动端模组历史中具有代表性意义。当前其主要价值是旧项目维护与技术史研究，不适合作为新项目的默认技术路线。
