# 中国版常见问题  <!-- md:flag china -->

本页汇总了中国版《我的世界》开发过程中的常见问题和解答。

## MC Studio与开发工作台

### 关于MC Studio

**Q: MC Studio是什么？**

A: MC Studio是中国版官方提供的集成开发工具，包含开发者启动器、地图编辑器、关卡编辑器、逻辑编辑器、特效编辑器和云端测试平台。

**Q: 如何下载MC Studio？**

A: 访问[中国版官网](https://mc.163.com/dev/process.html#xz)下载MC Studio。需要使用开发者账号登录使用。

**Q: MC Studio支持哪些操作系统？**

A: 目前支持Windows（建议Windows 10或更高版本）。Mac和Linux版本未来可能支持。

**Q: MC Studio占用多少硬盘空间？**

A: 基础安装约1-2GB，加上缓存文件可能需要3-5GB。建议预留足够的硬盘空间。

### 常见技术问题

**Q: 启动MC Studio时出错怎么办？**

A: 
1. 检查网络连接
2. 确保已登录开发者账号
3. 尝试卸载重装
4. 检查系统日志或查看官方反馈途径

**Q: 作品上传速度很慢怎么办？**

A: 
1. 检查网络连接和上传速度
2. 确保没有其他占用带宽的程序
3. 尝试更换网络
4. 联系技术支持

**Q: 如何在多台电脑上使用同一个开发者账号？**

A: 用同一个开发者账号登录MC Studio即可。但注意不要同时在多台电脑上修改同一个作品，避免版本冲突。

## 编辑器相关

### 地图编辑器

**Q: 如何创建新的地图？**

A: 在MC Studio中点击"新建基岩版组件"，选择"空白地图"，进行基本设置后即可创建。

**Q: 地图编辑器有大小限制吗？**

A: 地图尺寸通常限制在一定范围内。创建过大的地图可能导致性能问题。

**Q: 如何导出地图？**

A: 在MC Studio中选择要导出的作品，点击"导出"按钮，选择导出位置和格式。

### 关卡编辑器

**Q: 关卡编辑器用来做什么？**

A: 用于创建和编辑游戏关卡、设置流程、添加触发器等。

**Q: 如何设置关卡通过条件？**

A: 在关卡编辑器中使用触发器和事件系统，设定通过条件（如击败敌人、收集物品等）。

### 特效编辑器

**Q: 特效编辑器支持哪些效果？**

A: 支持粒子效果、模型特效、序列帧动画、文字面板等多种效果。

**Q: 如何在作品中使用自定义特效？**

A: 在特效编辑器中创建特效，然后在地图或逻辑编辑器中调用该特效。

## 网络游戏开发

### 开服工具

**Q: 如何使用网络游戏开服工具？**

A: 
1. 在MC Studio中新建网络游戏项目
2. 使用开服工具进行服务器配置和部署
3. 在云端测试平台进行联网测试

**Q: 开服工具支持的最大玩家数是多少？**

A: 根据不同配置，通常支持1-100+玩家同时在线。具体数值请查看文档。

**Q: 网络游戏如何进行开发阶段的热更新？**

A: 在开发者协作成员设置中勾选"开发阶段允许配置、部署与热更"权限，即可进行热更新。

**Q: 如何调试网络游戏中的Bug？**

A: 可以使用MC Studio的云端测试平台进行网络环境下的调试。上线后可使用控制台调试功能。

## 组件相关

### 组件上传

**Q: 上传组件时文件大小有限制吗？**

A: 有。单个组件通常不超过500MB。超大组件可能无法上传或审核通过。

**Q: 支持的组件格式有哪些？**

A: 支持.mcpack（组件包）和.mcworld（世界模板）格式。

**Q: 如何正确打包组件？**

A: 使用MC Studio的导出功能，选择"组件包"或"世界"格式，MC Studio会自动进行打包。

### Python模块

**Q: 为什么我的Python脚本无法运行？**

A: 检查使用的Python模块是否在官方白名单中。只有白名单内的Python模块才被允许使用。

**Q: 哪些Python模块被允许使用？**

A: 官方维护了一份Python模块白名单。详见审核规范文档。

**Q: 如何申请新增Python模块到白名单？**

A: 向官方开发者反馈提交申请，说明用途和必要性。

## 数据和统计

### 作品数据

**Q: 如何查看作品的下载量和评分？**

A: 登录开发者平台，进入"数据分析"，可以查看各项数据统计。

**Q: 评分是如何计算的？**

A: 根据玩家的评分汇总计算。可以在作品页面查看具体的评分分布。

**Q: 收益何时到账？**

A: 通常每月结算一次，次月进行出金。具体时间请查看官方说明。

## 错误排查

### 打包错误

中国版提供了详细的打包错误信息汇整，包括：

- 文件格式错误
- 清单文件错误
- 纹理加载错误
- 脚本语法错误
- 性能超标错误

遇到打包错误时，根据错误码查看对应的解决方案。

### 常见错误信息

**Error: 清单文件格式错误**

检查manifest.json文件的JSON语法是否正确。

**Error: 性能超标**

检查作品的帧率、内存占用、包体积等性能指标。

**Error: 文件缺失**

确保所有必要的文件都已包含，特别是纹理文件和脚本文件。
