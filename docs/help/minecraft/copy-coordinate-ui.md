# 复制坐标UI

复制坐标UI是Minecraft基岩版为创作者提供的坐标复制功能。它可以把玩家当前位置或准星所指方块的位置复制到剪贴板，并在聊天栏、命令方块等位置粘贴，适合快速编写传送、填充、结构和测试命令。

## 启用

1. 打开Minecraft。
2. 进入**设置**。
3. 打开**创建者**页。
4. 启用**启用复制坐标UI**。

启用后，聊天界面顶部会显示坐标选择和复制按钮。键盘、鼠标、手柄和触控平台的显示形式可能不同，但功能目标一致。

## 复制坐标

复制坐标时可以选择两类位置：

- **本地**：玩家当前位置。
- **准星**：准星所指方块的位置。

在键盘上，默认快捷键通常为：

| 操作 | 默认快捷键 |
|---|---|
| 复制玩家当前位置 | ++ctrl+alt+c++ |
| 复制准星所指位置 | ++ctrl+alt+x++ |

如果快捷键与系统或其他软件冲突，可以在**键盘与鼠标**设置中重新绑定。旧教程中可能出现其他快捷键说明，应以当前版本设置界面中显示的绑定为准。

## 粘贴坐标

复制后的坐标可以粘贴到聊天栏、命令方块输入框或外部文本编辑器中。控制器和触控界面通常会在相关输入区域提供粘贴按钮；电脑上也可以使用系统粘贴快捷键。

粘贴结果是不带逗号的坐标文本，例如：

```text
53 70 35
```

在命令中使用时，需要根据命令语法补齐命令名、目标选择器和其他参数。
