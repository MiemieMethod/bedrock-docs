# 账号登录与多人权限

在Minecraft基岩版中，账号登录问题与多人权限问题经常会表现为同一种现象，例如无法进入服务器、Realm不可见、提示禁止联机，或在“搜索玩家”时直接报错。排查时应先区分“账号未正确登录”“账号未拥有游戏”和“账号没有多人权限”三类原因。

## 确认登录的是正确账号

应首先确认当前登录的Microsoft账号就是购买Minecraft的账号。可通过以下方式核对：

- 检查Minecraft购买确认邮件发送到的邮箱地址。
- 登录Microsoft账号页面，查看订单历史中是否存在对应的Minecraft购买记录。

若账号本身的密码、验证方式或登录信息需要重设，应先在Microsoft账号服务中处理，而不是直接反复重装游戏。

## 先登录Xbox应用

在Windows设备或其他需要Microsoft账号联动的环境中，若游戏内登录异常，可先登录Xbox应用，再重新进入Minecraft。

1. 安装并打开Xbox应用。
2. 使用拥有Minecraft的Microsoft账号登录。
3. 核对该账号对应的头像、玩家代号和邮箱是否正确。
4. 重新启动Minecraft，再次尝试登录。

如果Xbox应用中显示的账号信息本身就不正确，应改用“使用其他账号登录”切换到正确账号。

## 儿童账号的多人权限

儿童账号即使已经拥有Minecraft，也可能因为家庭安全设置而无法联机。Microsoft与Xbox的家庭安全系统会控制以下能力：

- 加入多人游戏或Realm。
- 进行跨网络联机。
- 添加好友与关注其他玩家。
- 在多人游戏中使用文字、语音或邀请通信。

这些权限可通过Xbox在线隐私设置页或Xbox Family Settings应用调整。权限变更后，通常需要退出Minecraft并重新启动，新的设置才会生效。

## 平台订阅与联机前提

对于基岩版，多人游戏和Realm通常还依赖以下条件：

- 设备安装的是Minecraft基岩版，而不是其他版。
- 所有参与者使用彼此兼容的版本。
- 主机世界已启用多人游戏。
- 在主机平台上联机时，账号具备对应的在线服务订阅，例如Xbox Game Pass Core、Nintendo Switch Online或PlayStation Plus。

如果问题只发生在多人模式，而单人模式正常，应优先检查这些联机前提，而不是先假定世界文件已损坏。

## 无法加入多人游戏时的排查顺序

1. 确认本机与对方都已更新到目标版本，不要把正式版与预览版混用。
2. 确认登录账号拥有游戏，且游戏内显示的账号与订单历史一致。
3. 确认儿童账号、跨网络联机和好友权限已开启。
4. 确认主机世界开启了多人游戏，或服务器地址、端口填写正确。
5. 确认网络连接稳定，并检查自定义DNS、端口限制或防火墙设置是否拦截了Minecraft联机。
