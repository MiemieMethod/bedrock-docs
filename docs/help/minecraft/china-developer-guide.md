# 中国版开发者指南  <!-- md:flag china -->

本指南汇总了我的世界中国版特有的开发者相关内容，包括开发者账号管理、上架审核、团队协作等方面的说明。

## 注册与账号管理

### 成为开发者

注册成为中国版开发者需要准备以下材料：

- 1个网易邮箱账号（推荐163邮箱，126邮箱无法被添加为协作成员）
- 1个手机号码
- 本人身份证号和姓名
- 1个自己的QQ号

银行卡认证需要：

- 本人有效身份证
- 本人开户的银行卡（必须为本人名字的I类账户）

前往[中国版官方网站](http://mc.163.com/)点击【成为创造者】按钮进入注册页面，或[直接进入开发者平台](https://mcdev.webapp.163.com/#/login)。

### 开发者等级系统

中国版开发者等级系统由三个模块组成：

#### 开发者基础等级

采用经验值累计升等的方式，玩家在平台内的购买行为会转化为开发者的经验分。等级分为：

- **元气新星**（一~四级）：初级开发者
- **巧手工匠**（一~四级）：中级开发者，升级需要完成指定考核任务
- **杰出精英**（一~四级）：高级开发者，升级需要完成指定考核任务

#### 月度能力贡献

以月为单位统计开发者在手机版和电脑版的双端内容贡献，开发者将根据分值在排名榜上排名。

#### 信用分

信用是开发者永续经营的基础。开发者应自觉遵守规范，违反规则会导致信用分被扣分。

### 多账号协作

通过账号管理中的【协作成员管理】功能，可以添加协作成员账号。协作成员需要经过实名认证才能使用协作功能。

可以为成员设置的权限包括：

- 测试版启动器登录权限（最多10个子账号，未过审开发者仅2个）
- 协作开发权限（访问云端组件、网络服等）
- 开发阶段允许配置、部署与热更
- 上线阶段允许控制台调试

## 开发者规范

### 开发者协议

所有开发者需要在平台内查看并同意开发者协议，该协议为开发者创造良性发展的市场环境，帮助开发者展现创意、获取收益。

### 开发者文明守则

自2024年11月14日起正式生效，开发者需要遵守以下规范：

#### 禁止的不文明行为

**侵权与抄袭**

- 搬运有版权的组件，未经授权引用或改编他人作品
- 素材过多使用原版贴图或与同类型组件过度相似
- 抄袭他人组件的代码或首创内容（人物形象、场景设计、艺术风格等）
- 抄袭他人内容素材中的首创部分（介绍视频、介绍图片、名称、描述、关键词等）

**恶性价格战**

- 通过竞相降低相似内容价格进行不正当竞争
- 初期免费上架后恶意调高价格，扰乱市场秩序

## 内容上架与审核

### 上架流程

中国版内容上架需要经过以下步骤：

1. 完成内容开发
2. 提交上架申请
3. 进行平台审核
4. 通过审核后发布

### 审核标准

内容审核涉及合规性、原创性、质量等多个方面。具体审核标准请参考平台发布的审核指南。

### 下架与申诉

审核不通过或违反规范的内容将被下架。开发者可以对下架决定进行申诉。

## 平台功能

### 开发者工作台

《我的世界》开发工作台是中国版的官方开发工具，提供：

- 资源编辑功能
- 组件上传管理
- 网络游戏开服工具
- 团队协作功能

### MC Studio

MC Studio是中国版提供的集成开发环境，支持本地组件开发和测试。
