# Minecraft美术风格

Blockbench常被用于制作接近Minecraft原版视觉语言的模型和纹理。这里的“风格”不是游戏引擎强制执行的规则，而是一组帮助资源看起来更统一的检查标准。

## 模型先概括形体

Minecraft风格通常用少量元素概括形体，再把细节交给纹理。能用一个立方体表达的体积，不应拆成大量小立方体；能用透明纹理表达的薄片、尖角或花纹，也不一定要建成真实厚度。

旋转元素可以用来表达斜面，但不应为了模拟圆形而堆出大量阶梯状旋转块。圆柱体、球体和曲面通常需要被概括成更简洁的方块结构。

## 保持像素比例

UV映射决定纹理如何贴到模型表面。制作原版风格资产时，应尽量保持模型单位与纹理像素之间的稳定比例。一个表面如果在模型上是`8×8`单位，纹理上也应使用对应比例的像素区域。

混合分辨率会破坏风格统一。例如，同一模型的一部分使用16像素风，另一部分使用64像素细节，会让资源显得像从不同画风中拼接出来。需要更高分辨率时，应让整个资产的纹理密度保持一致。

## 颜色与调色板

纹理通常从中间色开始，再加入阴影色和高光色。色阶不应只改变明度，也可以适度改变色相和饱和度，使材质更自然。常见流程是：

1. 用中间色确定轮廓和大面积区域。
2. 添加一个主要阴影和一个主要高光。
3. 根据材质需要补充更暗或更亮的色阶。
4. 最后检查像素簇、边缘和可读性。

避免无意义噪点。噪点不会自动产生材质细节，过多随机像素反而会让远处轮廓变脏。

## 常见像素问题

/// define
枕形阴影

- 从边缘向中心逐层变亮的阴影方式。它会让平面看起来鼓起，通常不适合Minecraft风格。

条带

- 像素沿直线或斜线排成明显阶梯，使观察者注意到像素网格，而不是材质形状。

锯齿

- 斜线或曲线的步长不一致，导致边缘出现突兀折角。

过度抖动

- 在不需要过渡的区域使用棋盘格或随机交错像素，使纹理显得脏乱。

## 方块、物品和实体

方块纹理需要检查平铺效果。可以用`3×3`方块墙预览纹理是否出现重复痕迹。早期只用两三档颜色即可检查平铺，不必等全部细节画完才发现图案重复。

物品纹理通常限制在`16×16`网格中。轮廓应先保证识别度，再添加来自左上方的高光和来自右下方的阴影。小物品可以牺牲真实比例来换取可读性。

实体纹理常使用盒式UV。通常上表面和正面应比底面和背面更亮，以帮助模型在游戏光照中保持体积感。绘制实体时，应经常切换到动画姿势或隐藏遮挡部件，检查运动后会暴露的面。
