## 使用指南

- 你可以使用Blockbench标志来致谢或链接到Blockbench。
- 不要将Blockbench标志或名称用作或作为你自己的标志或名称的一部分。
- 不要以看起来可能与官方Blockbench关联的方式使用Blockbench标志或名称（如果它确实与Blockbench关联的话）的产品或项目。
