# 制作Minecraft风格标题

这一页做一个可用于封面图、宣传图或教程配图的Minecraft风格标题。它不直接导入游戏，而是利用Blockbench的立方体、像素纹理和截图功能做一个“看起来像Minecraft资产”的三维标题。

## 先定标题规格

开始之前，先把标题当成一个像素美术项目来规划：

- 每个汉字或字母都要能在远处读清楚。
- 立方体数量尽量少，轮廓由模型负责，细节由纹理负责。
- 不要混用多种像素密度。标题主体如果按16像素风绘制，阴影和装饰也应保持同一尺度。
- 颜色先准备一组中间色，再准备阴影色和高光色。

/// tip | 适合练习的标题
第一次练习时可以只做两个字，或只做`BEDROCK`这类直线较多的英文标题。复杂汉字需要更多取舍，不要一开始就追求完整笔画。
///

## 新建模型

1. 新建一个`Generic Model`或`Bedrock Model`工程。前者更适合导出展示用三维文件，后者更接近Minecraft资源包工作流。
2. 把网格吸附保持在1单位或更粗的步长，避免出现半像素风格的细碎元素。
3. 为每个字建立一个骨骼或分组，例如`title_left`、`title_right`。
4. 用立方体拼出文字正面。厚度建议先设为`1`到`2`单位，太厚会抢走正面轮廓的可读性。

## 做出方块感

Minecraft风格标题通常靠三个层次成立：

1. **主体面**：最亮、最干净，负责读字。
2. **侧面厚度**：比主体面暗，用来表现三维挤出。
3. **投影或描边**：比侧面更暗，用来把标题从背景里分离出来。

你可以复制主体立方体，整体向后下方移动一小段，改成暗色，作为标题投影。这样比给每个笔画单独加阴影更快，也更容易保持一致。

## 绘制纹理

1. 创建纹理模板。
2. 在**Paint**模式先铺主体色。
3. 给上边缘和左上角加高光。
4. 给右下边缘加阴影。
5. 检查是否出现“枕形阴影”：如果每个面都是中心亮、边缘暗，标题会显得臃肿。像素风更适合明确的受光方向。

/// warning | 不要过度抗锯齿
Minecraft风格允许清晰的像素边界。标题如果大量使用柔边、半透明和混合分辨率，放进像素风页面时反而会显得不统一。
///

## 设置展示角度并导出图片

1. 在预览窗口右键，选择合适的预设角度，或手动旋转到略微俯视的角度。
2. 关闭不需要的网格和辅助线。
3. 使用截图功能导出透明背景图片。Blockbench默认快捷键可通过**Preferences**→**Keybindings**查看，文档中也提到++ctrl+p++可用于截图。
4. 如果需要给文档配图，把图片放到与页面对应的资源目录，再在页面中引用。

## 保存源文件

导出的图片只是成品。请把`.bbmodel`源文件保留下来，后续修改文字、颜色、角度时会节省大量时间。
