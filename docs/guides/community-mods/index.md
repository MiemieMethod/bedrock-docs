# 社区模组系列教程

这里收录旧版或社区维护的基岩版客户端模组生态。它们通常不属于Mojang当前官方支持的附加包体系，也不等同于服务器插件。阅读这一部分时，请先接受一个现实：很多旧生态依赖特定Android版本、特定Minecraft版本或已经停止维护的加载器，更适合作为历史资料、旧模组维护资料或迁移参考。

## 什么时候还需要读这一组

- 你要维护旧Android客户端模组。
- 你要把旧ModPE或ICMod思路迁移到附加包、脚本API或服务器插件。
- 你要理解基岩版早期社区模组如何处理物品、方块、回调、资源和构建。

如果你是从零开发新内容，优先考虑官方附加包、国际版脚本API、中国版模组SDK或服务端插件。旧客户端模组的兼容成本通常更高，发布渠道和玩家环境也更难控制。

## 资料完整性说明

知识库中有Nernar维护的InnerCore开发文档，包含模组结构、物品、方块、回调、维度、界面和发布等内容。知识库中没有检索到独立、系统的ModPE教程资料，只检索到InnerCore文档中对BlockLauncher和ModPE的比较、替代和兼容提及。因此，ModPE页面会明确标出资料缺口，不会编造具体API教程。
