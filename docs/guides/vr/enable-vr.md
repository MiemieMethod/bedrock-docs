# 启用VR模式

在Windows上启动Minecraft基岩版VR时，最直接的做法是使用`minecraft://Mode/?OpenXR=true`这个启动入口。只要头戴设备、驱动和OpenXR运行时已经准备好，游戏就可以尝试直接进入VR模式。

/// warning | 适用范围
本页主要面向Windows环境。不同主机平台的VR启动方式并不相同，不要直接照搬这里的步骤。
///

## 创建快捷方式

1. 在桌面空白处右键。
2. 选择“新建”→“快捷方式”。
3. 在目标位置输入`minecraft://Mode/?OpenXR=true`。
4. 为快捷方式起一个便于辨认的名字，例如“Minecraft VR”。
5. 保存快捷方式。

这样一来，桌面上就会出现一个专门用于进入VR模式的启动入口。

## 启动前先检查什么

在双击快捷方式之前，建议先确认以下几项：

- 头戴设备已经连接并能被系统识别。
- 当前OpenXR运行时已经设置正确。
- 系统层面的VR环境已经完成初始化。

如果这些前置条件没有准备好，游戏通常不会按预期进入VR模式。

## 启动游戏

1. 打开头戴设备对应的平台软件或运行时环境。
2. 双击刚才创建的快捷方式。
3. 等待Minecraft启动并检测VR环境。
4. 进入游戏后确认画面已经切换到VR显示方式。

/// tip | 排查顺序
如果没有进入VR模式，优先检查头戴设备连接、OpenXR运行时和系统VR设置，再回头检查游戏入口本身。大多数问题都出在运行环境没有准备好，而不是快捷方式写错。
///
