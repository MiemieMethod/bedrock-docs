# 示例与DEMO旧版目录


## 示例索引页

- `4-DEMO示例/示例简介`

该页汇总了旧版中国版文档中提到的主要Demo名称，并把它们和对应教程建立了链接关系。旧版示例索引覆盖的主题包括：

- 入门脚本示例；
- 自定义物品、方块、生物、维度与传送门；
- 自定义状态效果、自定义音乐与远程武器；
- UI示例、联机大厅商品示例与虚拟世界示例；
- 微缩方块示例。
