# 过时教程

过时教程收纳仍有兼容性价值、但不应混入最新版主线的内容。这里的“过时”不等于“无用”：很多旧格式仍会被游戏接受，旧项目也需要维护；只是新项目不应该再把它们当作默认方案。

/// warning | 新项目优先阅读最新版教程
如果你正在创建新附加包，请先阅读最新版主线教程。只有在维护旧包、迁移旧包或分析历史资源时，才从本目录开始。
///

## 如何判断该不该读过时教程

适合阅读的情况：

- 你接手了旧资源包或行为包。
- 内容日志提示某个旧格式字段仍被读取，但推荐迁移。
- 你要解释为什么某些文件的`format_version`看起来比当前游戏版本低。
- 你要把旧教程改写为最新版教程。

不适合阅读的情况：

- 你只是想从零开始做新包。
- 你希望获得最新API用法。
- 你正在编写市场内容或需要严格遵守当前平台版本建议。

## 迁移原则

1. 先备份旧包。
2. 先让旧包在当前游戏中原样运行，记录内容日志。
3. 一次只迁移一种文件类型。
4. 每迁移一次就重新进游戏测试。
5. 对仍被官方当前指南认可的低格式版本，不要为了“数字好看”强行改高。
