## 核心概念

SerenityJS将服务器逻辑分为以下几个主要部分：

- **命令系统** - 通过命令系统处理玩家的文本命令输入
- **方块系统** - 定义和管理自定义方块类型、置换和行为
- **事件系统** - 响应游戏中发生的各种事件
- **世界管理** - 创建、加载和管理游戏世界

本教程系列将逐步深入这些核心概念。
