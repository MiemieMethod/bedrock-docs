# 使用指南

!!! note "关于此页面"
    本文整理自LeviLamina官方使用指南。若与原文存在冲突，以原文为准。

## 目的

我们非常喜欢用户和粉丝做出有趣的内容，并与社区分享。我们的目标是创造一个环境，让您可以为LeviLamina社区做出出色贡献，同时确保他人不会破坏这一体验。我们也理解，您创作和分享的内容中，有些是面向社区的，有些则可能是您自行创作并出售的。因此，我们制定了这些指南，用于说明在创作中如何使用LeviLamina的名称、品牌和资产。

简而言之，这些指南旨在帮助您理解可以做什么和不能做什么，帮助您创造并分享更多内容，同时限制那些越界行为。

## 重要法律协议

[LeviLamina最终用户许可协议](https://github.com/LiteLDev/LeviLamina/blob/HEAD/EULA.zh.md)是您与我们之间的法律协议。该协议涵盖您对LeviLamina产品的使用，并规定了您如何使用我们的网站，以及我们如何解决争议。您有责任理解并遵守该协议。

这些指南应与最终用户许可协议一并遵守，但不构成最终用户许可协议的一部分。它们存在的原因，是我们当前认为允许LeviLamina在某些方面保持一定灵活性是合理的。

对于所有用途（无论是否允许）：

- LeviLamina的名称、品牌、资产及其任何衍生物的所有权利（包括版权、商标权和相关权利）均归LeviMC所有，并将始终保留。
- 所有许可与同意都由我们自行决定，并可在我们认为适当时，或在我们不认可您的行为时随时撤销。
- 所有权利均明确保留。

如果您有意与我们建立正式合作关系，请保持该意向，因为我们目前不考虑与其他个人、组织或公司开展合作。如果您希望举报违反这些指南的行为，可以通过[我们的电子邮件](mailto:feedback@levimc.org)向我们发送信息。我们无法就某个具体项目是否符合这些指南提供建议。如果您不确定，应咨询律师获取帮助。

随着我们决定何时以及如何增加更多指南，欢迎您提出请求。

## 基本指南

我们希望明确，这些是适用于名称、品牌和资产的基本禁止性指南。您仍然应该完整阅读所有政策与指南，但这里先给出简要总结，帮助您抓住重点。

如果您使用了名称、品牌或资产的任何部分，那么：

**不要**从事任何可能让人误以为您分享的内容属于官方、已获批准、已获支持、与我们有关联，或由我们支持的活动，也不要在内容中包含此类暗示。

- **不要**从事任何非法、误导、猥亵、有害或令人反感的活动。
- **不要**从事任何可能损害或削弱我们名称、品牌或资产的活动，例如赌博、色情、暴力、恐怖主义或其他不安全、成人内容。
- **不要**重新分发我们的产品，或任何修改后的产品及产品文档。
- **不要**允许对我们创建的任何内容进行不公平或不合理的访问。
- **不要**冒充LeviMC，或表现得像是与LeviMC有关联，或者得到了LeviMC支持，并且应明确说明：
  - 您（而不是我们）对产品或服务负责，包括活动、服务器、书籍和手工制品。
  - 谁是出版者、制造者、卖家、组织者和/或所有者。
  - 产品、服务或任何相关购买的联系方式和联系信息。聊天和论坛链接不可接受。
- 不要通过转储LeviLamina或其他任何产品，尝试获取违反Minecraft最终用户许可协议的数据，或者可能危害Minecraft安全的数据。
- **不要**出于恶意目的对LeviLamina进行逆向工程。
- 我们不希望您使用LeviLamina开发任何可能危害Minecraft安全的程序，也不希望您使用LeviLamina开发任何我们认为会危害Minecraft安全的其他产品。

在您的产品、列表、说明、网站、页面以及所有相关材料中，请在显著位置标明免责声明：“这不是LeviMC的官方产品［产品/服务/活动等］。未获LeviMC批准，与LeviMC无关联。”

这些指南旨在帮助您理解，当您使用我们的名称、品牌和资产时，我们希望不会发生什么。我们尽量保持开放、诚实，最重要的是值得信任。我们也希望您以同样的方式对待我们。

## 命名指南

只要遵守本节的指南，您就可以在产品、服务、标题或列表中使用我们的名称，包括网站、视频平台或商品。

如果您：

- 将LeviLamina名称用作副标题、辅助标题或描述，因为这对真实、准确地描述您的创作或其用途是必要的。
- 确保包含LeviLamina名称的副标题不是完整名称或标题中最主要、最醒目的部分。
- 不将我们品牌或资产的其他部分作为相关品牌的一部分，包括作为徽标或徽标的一部分。
- 不将我们的名称用作与之无关的产品关键词或搜索标签，也不要将其用于侵权或仿冒产品。

那么，您可以使用LeviLamina名称。

## 个人使用

对于您为自己创作的内容，我们相当宽松。几乎所有事情都可以做，所以尽管去尝试、去享受吧——只要记得遵守政策，并避免任何非法活动，例如编写作弊程序、侵犯他人权利，或违反Minecraft最终用户许可协议。

如果您决定将作品分享给社区，您当然可以这样做。事实上，我们会为您感到高兴，也很期待看到您的作品。请注意，一旦您决定把内容分享给社区，无论是否收费，我们都将其视为商业活动。当您进行商业活动时，必须遵守商业使用指南。

例如，如果您想搭建并运营非商业博客、服务器、社区论坛、粉丝网站、粉丝俱乐部、新闻组、活动或聚会，这些规则同样适用。

## 商业使用

对于使用我们的名称、品牌和资产进行商业用途，我们的要求更严格。我们很高兴您愿意与社区分享创作，也理解您可能希望对作品收费，但对于LeviLamina的商业使用，仍然有一些限制。

在这些指南中，商业使用是指您使用并与他人分享我们的名称、品牌或资产的任何行为，无论您是否收费，或者是否免费提供。这可以体现在您制作的视频和直播、您制作的商品，甚至是您托管的服务器中。我们可能会随着时间调整这一定义，但目前它足以帮助您理解什么可以做、什么不可以做，从而帮助您和其他粉丝创作更多内容，同时限制那些过度越界的行为。

目前，我们遵循Minecraft使用规则中的商业使用规定。

## 总结

我们很高兴看到您作为社区的一员，去做有趣的事情，并希望支持您的创造力。我们理解，您可能希望与他人分享创作，甚至通过创作获得收益。我们希望赋予您创作和分享的能力，同时也希望限制那些走得太远的行为。希望这些指南能帮助您理解我们允许什么，以及我们不允许什么。

如果某些内容没有被这些指南涵盖，而且我们没有明确说明可以做，那通常意味着我们不希望您这样做。无论如何，如果某项内容没有被涵盖，请不要在没有我们书面许可的情况下去做。如果某项内容已被这些指南、我们的账户条款，或适用于任何游戏的最终用户许可协议明确允许，那么您无需再与我们联系。
