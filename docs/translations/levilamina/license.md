LeviLamina的非闭源部分采用GNU宽通用公共许可证第3版（GNU Lesser General Public License Version 3，LGPL-3.0）发行。

## 适用范围

LeviLamina仓库README同时声明，项目版权归LeviMC所有，且仅“非闭源部分”按LGPL-3.0发行。这意味着：

- LeviLamina仓库中的开源部分以LGPL-3.0为主要许可依据。
- LeviMC的闭源软件不适用本页所述许可边界，而应另行参阅[最终用户许可协议](eula.md)。
- 名称、品牌、资产以及再分发边界，还需要结合[使用指南](usage-guidelines.md)理解；这些内容不是LGPL-3.0正文的一部分。
