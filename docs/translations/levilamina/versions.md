## 阅读说明

- `客户端版本`列为“无”表示该支持表在对应阶段只列出服务端兼容关系，并不表示任意客户端都可自由混用。
- 从表中可以看出，`1.8.x`、`1.9.x`与`26.10.x`已经重新给出客户端版本对应关系；安装客户端侧LeviLamina时，应优先核对这一列。
- 版本号匹配只解决“能否启动或连接”的前提问题；是否需要升级，还应结合[更新日志](changelog.md)中的崩溃修复、漏洞修复和运行时数据更新一起判断。
