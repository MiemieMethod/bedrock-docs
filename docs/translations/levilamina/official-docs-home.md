## 首页定位

文档首页将LeviLamina描述为轻量级、模块化、多功能的基岩版模组加载器，并明确其为LiteLoaderBDS的后继项目。

首页中的概述只给出高度抽象的项目介绍：

- LeviLamina是非官方模组加载器。
- 目标是为基岩版提供必要的API支持。
- 重点强调C++模组开发、事件系统、实用接口与封装后的开发基础设施。

这些表述与项目说明页中的定位基本一致，但文档首页更偏向站点入口，而不是仓库说明。
