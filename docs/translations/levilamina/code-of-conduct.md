# 行为准则


本页为LeviLamina仓库行为准则的中文整理版本。若与原文存在冲突，以原文为准。

## 我们的承诺

我们作为社区成员、贡献者和管理者，承诺让所有人都能在没有骚扰的环境中参与社区，不因年龄、身体外观、可见或不可见残疾、族裔、性别特征、性别认同与表达、经验水平、受教育程度、社会经济地位、国籍、外貌、种族、宗教或性取向而受到区别对待。

我们承诺以有助于建立开放、友善、多元、包容且健康社区的方式行动和互动。

## 我们的标准

能够为社区营造积极环境的行为包括：

- 对他人表现出同理心与善意。
- 尊重不同的意见、观点与经历。
- 给予并坦然接受建设性反馈。
- 对自己的错误负责，并向受影响者道歉，同时从中吸取经验。
- 关注的不只是个人利益，而是整个社区的最佳利益。

不可接受的行为包括：

- 使用性化语言或图像，以及任何形式的性暗示或性骚扰。
- 恶意取笑、侮辱、贬损性评论，以及私人或政治攻击。
- 公开或私下骚扰他人。
- 在未经明确许可的情况下公开他人的私人信息，例如实际住址或电子邮箱。
- 其他在专业环境中可合理视为不当的行为。

## 执行责任

社区负责人有责任澄清并执行这些可接受行为标准，并会针对其认定的不当、威胁性、冒犯性或有害行为采取适当且公正的纠正措施。

社区负责人有权且有责任移除、编辑或拒绝不符合本行为准则的评论、提交、代码、维基编辑、问题单及其他贡献，并会在适当时说明其审查决定的原因。

## 适用范围

本行为准则适用于所有社区空间，也适用于个人在公开场合代表社区发言的情形。代表社区的情形包括使用官方电子邮箱、通过官方社交媒体账号发帖，或作为指定代表参加线上或线下活动。

## 执行

任何形式的辱骂、骚扰或其他不可接受行为，都可以向负责执行本准则的社区负责人报告（原文未提供具体联系方式）。所有投诉都会被及时且公正地审查和调查。

所有社区负责人都有义务尊重举报人的隐私与安全。

## 执行指南

社区负责人会依据以下社区影响指南，判断某项行为违反本行为准则后的后果。

### 1. 纠正

**社区影响**：使用不当语言，或其他被视为在社区中不专业或不受欢迎的行为。

**后果**：由社区负责人私下发出书面警告，说明违规性质并解释其不当之处。可能要求公开道歉。

### 2. 警告

**社区影响**：单次事件或一系列行为构成的违规。

**后果**：发出警告，并附带持续违规将面临的后果。在指定期限内，不得与涉事人员发生任何互动，包括向执行本准则的人员发起未经请求的接触。这包括避免在社区空间以及社交媒体等外部渠道中的互动。违反这些条件可能导致临时或永久封禁。

### 3. 临时封禁

**社区影响**：严重违反社区标准，包括持续的不当行为。

**后果**：在指定期限内，禁止以任何形式与社区进行互动或公开交流。在此期间，不得与涉事人员发生公开或私下互动，包括向执行本准则的人员发起未经请求的接触。违反这些条件可能导致永久封禁。

### 4. 永久封禁

**社区影响**：表现出持续违反社区标准的模式，包括持续的不当行为、对个人的骚扰，或对某类群体的攻击或贬损。

**后果**：永久禁止以任何形式在社区中进行公开互动。
