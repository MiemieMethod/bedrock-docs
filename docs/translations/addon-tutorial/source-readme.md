# 来源前言摘录

本页保留来源`docs/README.md`中的核心说明，用于历史语境核对。

## 资料定位

来源将该系列定义为“Minecraft基岩版附加包开发教程三部曲”，分为《入门教程》《进阶教程》《登峰教程》三个篇章，分别面向初学者、有一定经验的开发者和偏研究向读者。

## 来源前言要点

- 附加包被定位为基岩版中以数据驱动为主的“原版模组”形式。
- 教程编写背景强调了附加包在实体、方块、物品、生物群系、地物与脚本方向的扩展能力。
- 资料写作时间跨度较早，部分接口命名、实验性开关和推荐路径已与当前版本存在差异，使用时需结合本站现行文档复核。
