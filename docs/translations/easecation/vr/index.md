# 虚拟现实

**虚拟现实（Virtual Reality）**是指借助头戴式显示设备与交互输入装置，使Minecraft以沉浸式方式运行的使用形态。在基岩版环境中，VR功能通常受平台、设备驱动和运行时支持共同影响，因此相关资料既涉及启动方式，也涉及资源包和模型适配。

本分区仅收录与Minecraft基岩版VR相关的内容：

- [启用VR模式](installing-bedrock-vr.md)
- [设置资源包](pack-setup.md)
- [编辑你的第一个模型](editing-your-first-model.md)
