# Blockception架构翻译

本目录包含Blockception维护的Minecraft基岩版JSON架构相关文档的翻译。

## 可用文档

- [**IDE设置指南**](ide-setup.md) - 在各种IDE和编辑器中配置JSON架构的完整指南，包括Neovim、Visual Studio Code、JetBrains IDE和Sublime Text。

## 关于Blockception

Blockception是一个社区项目，维护了全面的Minecraft基岩版JSON架构库，用于帮助开发者在开发附加包时获得代码补全、验证和错误检查。
