本页保留Apollo官方插件介绍页的原文分类脉络。原文说明：官方插件即公共Mod，建议在其基础上进行二次开发；旧版资料中可通过MC Studio中基岩版服务器分页的“官方插件”入口获取插件列表。

## 游戏玩法

- 领地插件：玩家专属区域与交互限制。
- 区块版领地插件：单区块零级领地、子领地、权限控制、区域预览与迁移。
- 私有箱子插件：专属箱子与权限申请。
- 战斗系统插件：RPG数值体系与扩展装备位。
- 宝石插件：装备镶嵌与培养系统。
- 随机传送插件：跨服、跨维度与本维度内安全随机传送。
- 回合战斗插件：角色、属性、技能与表现驱动的回合制战斗。

## 游戏系统

- 公告插件：邮件、登录弹窗、轮播公告。
- 队伍插件：组队、招募与入队申请。
- 任务插件：主线、支线、引导任务，并可复用关卡编辑器数据。
- 好友插件：好友关系、附近玩家与平台好友导入。
- 公会插件：公会创建与管理。
- 经济插件：多货币、商店与摆摊。
- 主菜单插件、喇叭插件、弹窗提示插件、称号插件、排行榜插件、PVP插件、弹幕插件、面板描述插件。

## 基础功能

- 地图属性插件：浮空字、边界、掉落物清理、流体与藤蔓控制。
- 云端玩家信息插件：跨地图背包同步。
- 功能NPC插件：转服NPC。
- 权限管理插件：玩家组与聊天前缀。
- 副本管理插件：单地图多副本调度。
- 唯一ID插件、飞行插件、聊天插件、聊天记录插件、面对面交易插件、外观管理插件。
- 报名匹配插件、通用输入界面插件、通用综合界面插件、通用显示界面插件。
- 数据转存插件、地图转存数据库插件。

## 活动和运营

- 活动奖励插件、每日登录奖励插件、商城插件、运营数据插件。
- 累计消费活动插件、定时指令插件。
- 反作弊插件：非法破坏、非法移动、刷物品与非法附魔检测。
- 操作记录插件：区域行为记录与回滚。
