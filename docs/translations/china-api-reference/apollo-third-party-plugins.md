## 目录摘要

原文列出的第三方插件包括：

- 抽奖插件、物品栏菜单插件、背包和箱子整理插件、随身仓库插件、图鉴插件、宝箱礼包插件。
- 货币插件、经验等级插件、小地图插件、定时刷新生物方块插件、任务插件、占位符插件、投票插件。
- 7stars坐骑插件、自定义分解插件、自定义合成插件、排队插件、第五季节核心插件、成就插件。
- 全息插件、定时清理实体插件、战令插件、生物血条插件、空岛插件。

## 原文强调的信息

1. 该目录用于展示生态中可下载的第三方插件，而非完整的接口手册。
2. 多数条目仅包含功能简介与作者署名，具体接入方式仍需依赖下载包内的说明文件。
3. 若插件暴露进一步的接口或服务能力，旧版资料通常不会在本页展开。
