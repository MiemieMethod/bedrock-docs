# 数据驱动目录总览

/// info | 来源
- 原文仓库：[`MiemieMethod/mcbe-dev-home`](https://github.com/MCBE-Development-Wiki/mcbe-dev-home)

该页面保留原始数据驱动目录的分类脉络，用于历史资料索引与旧文档术语对照。

## 原始分类结构

| 分类 | 原始列举重点 |
|---|---|
| 常规 | 清单文件 |
| 渲染相关 | 纹理、纹理集、着色器、材质、模型、相机 |
| 声音相关 | 音效、音乐 |
| 界面相关 | JSON UI、i18n、字体、表情符 |
| 实体相关 | 实体客户端定义、实体服务端定义、动画、动画控制器、渲染控制器、生成规则、生成组、行为树 |
| 方块和物品相关 | 方块客户端定义、方块服务端定义、物品客户端定义、物品客户端偏移量定义、物品服务端定义 |
| 效果相关 | 粒子、迷雾、功能域 |
| 世界生成相关 | 维度、生物群系客户端定义、生物群系服务端定义、地物、地物规则 |
| 表相关 | 战利品表、交易表、配方、NPC对话 |
| NBT相关 | 存档文件、LevelDB、结构 |
| 命令相关 | 函数、函数滴答 |

## 术语语境

原文将“数据驱动”定义为使用静态文件添加或修改游戏内容的开发方式，并将相关文件、组件与内容分别称为“数据驱动文件”“数据驱动组件”“数据驱动内容”。
