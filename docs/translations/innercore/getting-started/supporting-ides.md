原文档推荐使用可提供补全、诊断和构建能力的IDE，而不是纯文本编辑器。常见选择包括Visual Studio Code、IntelliJ IDEA以及移动端的Spck Code Editor。

## 推荐组合

- 以脚本为主：Visual Studio Code即可起步。
- 需要Java开发：可用Java扩展包或IntelliJ IDEA。
- 需要C/C++开发：可配合C/C++扩展或CLion。

## 工具链

Inner Core Mod Toolchain用于统一处理TypeScript编译、Java构建、C/C++构建和设备调试推送流程。对于跨语言项目，该工具链可显著减少环境搭建成本。

## 补充工具

除代码环境外，文件管理器、贴图编辑器、模型工具和数据编辑器同样是旧客户端模组开发的高频工具。
