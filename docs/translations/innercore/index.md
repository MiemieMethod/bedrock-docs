InnerCore是一个适用于Android设备上Minecraft基岩版的传统客户端模组框架，通过Horizon启动器运行。它在游戏原版功能之上提供了扩展接口，允许开发者使用JavaScript、TypeScript、Java及C++创建各类游戏修改内容（模组）。
