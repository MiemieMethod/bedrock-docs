**发布时间**：2026年5月21日

Android和Realm的新修复版本正在推出，以解决最新更新中发现的问题。由于需要在各个平台获得批准，所以请耐心等待该版本到达您的设备。

## 修复 {#fixes}

- 修复了一个问题，该问题在某些Android设备上启用了鲜艳视效时导致屏幕上出现黑色方块（[MCPE-238627](https://bugs.mojang.com/browse/MCPE-238627)）
- 修复了游戏期间可能出现的多个崩溃问题
