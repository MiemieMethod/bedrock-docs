**发布时间**：2026年5月14日

新的修复版正在逐平台推送，用于处理上一次更新中发现的问题。由于各平台审核节奏不同，更新时间可能存在延迟。

# 修复

## 游戏玩法

- 包含多项错误修复与稳定性改进。

## 用户界面

- 修复了在没有可用配方时打开熔炉导致卡死的问题。

## 稳定性与性能

- 修复了多个游戏过程中可能出现的崩溃。
- 修复了一个启动阶段可能出现的崩溃。
- 修复了未登录状态下点击“搜索玩家”按钮导致崩溃的问题（[MCPE-235488](https://bugs.mojang.com/browse/MCPE-235488)）。

# 技术更新

## 方块

- 修复了在`format_version >= 1.26.20`时，方块特质未正确遵循实验性要求的问题。
