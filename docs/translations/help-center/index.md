# 官方帮助中心翻译

基岩文档项目对Minecraft官方帮助中心的内容进行了翻译和归档。本部分包含来自官方支持网站的主要资源的中文版本。

## 更新日志

该分类包含所有Minecraft基岩版（国际版）的官方发布日志，从版本1.0至最新版本的完整历史。

### 最新更新

最新的版本日志可以在下方找到。每个日志包含该版本的所有新功能、改进和修复。

- [Minecraft：基岩版 26.23 修复版](Minecraft-Bedrock-Edition-26-23-Hotfix-Changelog.md)
- [Minecraft - 1.21.132 （基岩版）](Minecraft-1-21-132-Bedrock.md)

### 完整日志列表

该分类包含以下类型的官方发布日志：

#### 主要版本更新（Bedrock Edition）

包含所有Minecraft基岩版的主要版本和热修复版本日志。格式通常包括：

- **新增功能**（New Features）- 版本新增的内容
- **调整**（Tweaks）- 对现有功能的改进和调整
- **修复**（Fixes）- 错误和问题的修复
- **技术更新**（Technical Updates）- 对开发者相关的API和系统的更新

#### 测试版和快照（Beta & Snapshots）

包含测试版本的更新日志。详见Beta-and-Preview-Information-and-Changelogs分类。

#### 历史版本（Legacy Versions）

包含早期MCPE（Minecraft Pocket Edition）和其他遗留平台的日志。
