**发布于：** 2021年5月6日

**在参与Minecraft Beta之前，请阅读：**

- 加入测试版将用一个正在开发中的Minecraft版本替换您的游戏
- 您将无法访问领域，并且在预览测试版期间无法加入非测试版玩家
- 在测试版中玩的任何世界无法在之前的游戏版本中打开，因此请务必备份世界以防丢失
- 测试版构建可能不稳定，并且不代表最终版本的质量
- 测试版仅在Xbox One、Windows 10和Android（Google Play）上提供。要加入或退出测试版，请参见[aka.ms/JoinMCBeta](https://aka.ms/JoinMCBeta)以获取详细说明

又到了洞穴与山崖测试版更新的时刻！请继续在[aka.ms/CavesCliffsFeedback](https://aka.ms/CavesCliffsFeedback)的讨论中给我们反馈，并在[bugs.mojang.com](https://bugs.mojang.com/)搜索和报告您可能遇到的任何新漏洞。

## 洞穴与山崖 - 实验性特性

- 箱子矿车现在也可以出现在废弃矿井中
- 要塞在更大的洞穴中将更加隐蔽
- 修复了潜影贝在y=0以下无法传送的漏洞（[MCPE-121810](https://bugs.mojang.com/browse/MCPE-121810)）

## 洞穴与山崖 - 非实验性

### 美西螈

- 美西螈在假死状态下无法再自行移动（[MCPE-124128](https://bugs.mojang.com/browse/MCPE-124128)）
- 美西螈在陆地上不再假死

### 细雪

- 发射器现在可以使用细雪桶发射细雪块
- 骷髅在细雪中站立时不再被冻结，并将在45秒后变为流浪者（[MCPE-116781](https://bugs.mojang.com/browse/MCPE-116781)）
- 生物现在可以在细雪上正确寻路（[MCPE-118214](https://bugs.mojang.com/browse/MCPE-118214)）
- 原版趋同：细雪不再支持脚手架（[MCPE-120947](https://bugs.mojang.com/browse/MCPE-120947)）
- 原版趋同：细雪块的开采时间现在与Java版的开采时间相匹配
- 玩家不再可以通过跳跃来避免在细雪中下沉（[MCPE-112023](https://bugs.mojang.com/browse/MCPE-112023)）

### 杜鹃树叶

- 杜鹃树和盛开的杜鹃树叶在使用精准采集工具开采时现在会掉落自身

### 洞穴与山崖

- 在极端丘陵生物群系中可以找到寄生的深板岩变种
- 铜块现在可以用9个铜锭合成
- 铜块可以拆分为9个铜锭
- 铜矿石和深板岩铜现在掉落2-3个粗铜
- 粗铁、粗铜和粗金的物品纹理已更新
- 粗铁、铜和金块的纹理已更新
- 指南针和时钟的纹理已更新

### 孢子花

- 孢子花在水中不再存活

### 铜

- 铜在打蜡/去蜡时现在具有正确的图形和音效（[MCPE-121827](https://bugs.mojang.com/browse/MCPE-121827)）
- 当放置在铜台阶上时，树叶的底面现在正确渲染

### 发光鱿鱼

- 鱿鱼和发光鱿鱼在释放墨水时现在会发出特定的声音
- 旧的告示牌在染色时不再发光（[MCPE-117835](https://bugs.mojang.com/browse/MCPE-117835)）

### 山羊

- 通过将尖叫山羊与非尖叫山羊繁殖，获得尖叫山羊的几率增加

### 物品

- 修复了在使用望远镜缩放时第一人称副手物品被渲染的问题（[MCPE-125825](https://bugs.mojang.com/browse/MCPE-125825)）

### 小型滴水石锥

- 钟乳石在填充炼药锅时现在会更频繁地滴水

### 粗金

- 猪灵现在会仰慕粗金块
- 猪灵现在会仰慕粗金

### 缠根泥土

- 铺设缠根泥土不再立即将其转变为耕地块

### 望远镜

- 望远镜现在是创造模式物品栏中的“装备”类别的一部分（[MCPE-126079](https://bugs.mojang.com/browse/MCPE-126079)）

### 原版趋同

- 下落的方块在落在潜影贝上时现在会被破坏

### 用户界面

- 修复了控制器光标灵敏度缩放的问题（[MCPE-33868](https://bugs.mojang.com/browse/MCPE-33868)）
- 修复了阻止用户在“键盘和鼠标”设置中重新分配键'E'的问题（[MCPE-124224](https://bugs.mojang.com/browse/MCPE-124224)）

### 命令

- 修复了命令方块矿车未受'CommandBlocksEnabled'游戏规则影响的问题（[MCPE-93666](https://bugs.mojang.com/browse/MCPE-93666)）

### 多人游戏

- '/kick'命令在禁用作弊和命令方块时正常工作，并且不再使用目标选择器

# 技术更新

### 新的实验性特性：体积！

- 在您的世界中创建体积：只需在保存的世界中创建一个“volumes”文件夹，并在其中放置json文件 - 每个体积一个json文件。更多信息请参见测试版行为包文档
- 目前，您可以将迷雾 - minecraft或您自己的自定义迷雾 - 附加到体积。当玩家进入该体积时，迷雾将生效

一般

- 添加了包含体积实体信息的AddVolumeEntityPacket，包括其网络ID和包含体积定义和组件信息的NBT，这将被添加到客户端的注册表中
- 添加了RemoveVolumeEntityPacket，包含要从客户端注册表中移除的体积实体的网络ID

### 生物

- 向“minecraft:genetics”组件添加了一个新的可选标志"use_simplified_breeding"。当此标志设置为true时，后代永远不会继承父母的隐性等位基因作为主要等位基因，主要等位基因也不会作为隐性。这使得实现简单的繁殖逻辑变得更容易

### 游戏测试框架

- 组件
  - \[已移除\] 函数 setColor(color : Number)
  - 属性 value : Number - 获取或设置实体的颜色

  <!-- -->

  - 添加了额外的组件接口。运动、导航、可治疗、可呼吸、可繁殖和可驯化组件可以通过entity.getComponent(componentName : string)引用。
  - minecraft:color
- 容器 - 与实体容器交互的新接口。可以通过entity.getComponent("inventory").container引用
  - 函数 setItem(slot : Number, itemStack : ItemStack) - 将itemStack添加到给定槽位的容器中
  - 函数 getItem(slot : Number) : ItemStack - 获取给定槽位的itemStack
  - 函数 addItem(itemStack : ItemStack) - 将给定的itemStack添加到容器的第一个可用槽位
  - 函数 transferItem(fromSlot : Number, toSlot : Number, toContainer : Container) - 将itemStack从容器的fromSlot转移到toContainer的toSlot
  - 函数 swapItems(slot : Number, otherSlot : Number, otherContainer : Container) - 在容器的槽位和其他容器的其他槽位之间交换itemStack
  - 函数 worldLocation(relativeLocation : BlockLocation) - 将给定游戏测试位置的坐标转换为其对应的世界位置
  - 函数 relativeLocation(worldLocation : BlockLocation) - 将给定世界位置的坐标转换为其对应的游戏测试位置
