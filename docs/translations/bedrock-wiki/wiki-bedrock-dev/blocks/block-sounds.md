方块声音指的是在 `blocks.json` 中方块条目的声音属性。该属性用于确定一般的方块声音，例如挖掘声、踩踏声、破坏声和放置声。您可以通过以下方式为方块应用声音：

<CodeHeader>RP/blocks.json</CodeHeader>

```json
{
    "format_version": "1.21.40",
    "wiki:custom_log": {
        "sound": "wood" // 在此定义声音
    }
}
```
