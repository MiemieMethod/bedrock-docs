# /allowlist

> 文档版本：1.20.80.24

`/allowlist`命令Manages the server allowlist.

/// note | 执行条件
该命令需要权限等级：`internal`|`4`。该命令需要开启作弊。
///

## 用法

/// tab | `重载1`
```mcfunction
/allowlist <action:AllowListAction> [name:string]
```
