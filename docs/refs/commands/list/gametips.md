# `/gametips`

> 文档版本：1.20.80.24

`/gametips`命令Enable or disable the game tips on this device

/// settings | 执行条件
该命令需要权限等级：`any`|`0`。该命令需要开启作弊。
///

## 用法

/// tab | 重载1
```mcfunction
/gametips <status:Status>
```
