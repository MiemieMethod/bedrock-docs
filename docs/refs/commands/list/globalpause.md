# `/globalpause`

> 文档版本：1.21.0.20

`/globalpause`命令Sets or gets the paused state of the game for all players.

/// settings | 执行条件
该命令需要权限等级：`host`|`3`。该命令需要开启作弊。
///

## 用法

/// tab | 重载1
```mcfunction
/globalpause [isPaused:Boolean]
```
