# `/wsserver`

> 文档版本：1.21.0.20

`/wsserver`命令Attempts to connect to the websocket server on the provided URL.

/// settings | 执行条件
该命令需要权限等级：`admin`|`2`。该命令需要开启作弊。
///

/// info | 别名
该命令还可以使用以下别名：`/connect`。
///

## 用法

/// tab | 重载1
```mcfunction
/wsserver <serverUri:text>
```
