# `/getchunkdata`

> 文档版本：1.20.80.24

`/getchunkdata`命令Gets pixels for a specific chunk.

/// settings | 执行条件
该命令需要权限等级：`host`|`3`。该命令需要开启作弊。
///

## 用法

/// tab | 重载1
```mcfunction
/getchunkdata <dimension:Dimension> <chunkX:int> <chunkZ:int> <height:int>
```

//// html | div.result
///// define

  |`nether`||
  |`the_end`||


`chunkX`：<!-- md:samp int -->

`chunkZ`：<!-- md:samp int -->

- 基本类型。

`height`：<!-- md:samp int -->

- 基本类型。
