# `/getchunks`

> 文档版本：1.20.80.24

`/getchunks`命令Gets list of chunks that are loaded.

/// settings | 执行条件
该命令需要权限等级：`host`|`3`。该命令需要开启作弊。
///

## 用法

/// tab | 重载1
```mcfunction
/getchunks <dimension:Dimension>
```

//// html | div.result
///// define
`dimension`：<!-- md:samp Dimension -->

- 枚举类型。枚举值如下：

  |值|描述|
  |---|---|
  |`overworld`||
  |`nether`||
  |`the_end`||
