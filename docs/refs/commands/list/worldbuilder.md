# `/worldbuilder`

> 文档版本：1.21.0.20

`/worldbuilder`命令Toggle World Builder status of caller.

/// settings | 执行条件
该命令需要权限等级：`gamedirectors`|`1`。该命令需要开启作弊。
///

/// info | 别名
该命令还可以使用以下别名：`/wb`。
///

## 用法

/// tab | 重载1
```mcfunction
/worldbuilder
```
