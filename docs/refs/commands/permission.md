# `/permission`

> 文档版本：1.20.80.24

`/permission`命令Reloads and applies permissions.

/// note | 执行条件
该命令需要权限等级：`internal`|`4`。该命令需要开启作弊。
///

## 用法

/// tab | 重载1
```mcfunction
/permission <action:PermissionsAction>
```

//// html | div.result
///// define
`action`: <!-- md:samp PermissionsAction -->

- 枚举类型。枚举值如下：

|值|描述|
|---|---|
|`list`||
