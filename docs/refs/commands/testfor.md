# /testfor

> 文档版本：1.20.80.24

`/testfor`命令Counts entities (players, mobs, items, etc.) matching specified conditions.

/// note | 执行条件
该命令需要权限等级：`gamedirectors`|`1`。该命令需要开启作弊。
///

## 用法

/// tab | `重载1`
```mcfunction
/testfor <victim:target>
```
