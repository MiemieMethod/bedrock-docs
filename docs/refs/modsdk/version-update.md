## 使用建议

1. 先按版本号和日期定位目标更新段，再对照当前中国版参考页确认接口是否仍可用。
2. 同一版本下常有多次增量更新，排查行为变化时应按日期顺序阅读。
3. 历史条目中的“新增”并不等同于当前可用，最终以目标运行环境实测为准。
4. 涉及国际版脚本API时，需另查国际版文档，不应把本页索引外推到`@minecraft/server`体系。
