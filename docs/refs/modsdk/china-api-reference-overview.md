## 原文结构

### ModAPI

ModAPI是玩法开发的基础部分，主要收录系统、事件、组件和常用运行时接口。现在如果要继续查这些内容，可以直接看：

### Apollo

Apollo部分主要围绕网络服集群、服务器角色和配套插件展开。现在更适合直接从下面两页进入：

- [Apollo网络服](../../docs/server/apollo.md)
- [Apollo插件生态](../../refs/modsdk/apollo-plugin-ecosystem.md)

### PresetAPI

PresetAPI部分主要面向预设架构、预设对象和版本变更。继续查这部分内容时，可直接阅读：

- [中国版PresetAPI参考](../../refs/modsdk/preset-api.md)
