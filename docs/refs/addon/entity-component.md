# 实体组件

本页列出国际版官方实体参考中收录的实体组件标识符。实体组件写入行为包实体定义的`components`或`component_groups`对象，用于声明实体的属性、状态、交互、移动、战斗、事件感知和渲染相关行为。

/// note | 使用说明
`minecraft:behavior.*`类AI意向单独列于[实体AI意向](entity-ai-goal.md)。部分组件在原版中只服务于特定实体，或依赖硬编码行为；在自定义实体中使用时应以实际游戏测试结果为准。
///
