# 实体AI意向

本页列出国际版官方实体参考中收录的`minecraft:behavior.*`AI意向组件。AI意向是实体组件的一类，用于向生物加入寻路、目标选择、攻击、逃避、繁殖、工作、交易和特定生物行为。

## 执行优先级

多数AI意向使用`priority`字段控制调度顺序。数值越接近`0`，意向优先级越高；多个意向同时满足条件时，优先级更高的意向更可能先执行。部分意向还使用`control_flags`占用`move`、`look`、`jump`等控制通道，以避免互相冲突。
