# 清单模块架构参考

本文档对应Microsoft Learn的`minecraft:module`结构。

## 字段

| 字段 | 默认值 | 类型 | 说明 |
|------|--------|------|------|
| `description` | *未设置* | 字符串 | 模块描述。 |
| `entry` | *未设置* | 字符串 | 脚本模块入口路径。 |
| `language` | *未设置* | 字符串 | 脚本模块语言字段。 |
| `type` | *未设置* | 字符串 | 模块类型字段。 |
| `uuid` | *未设置* | 对象 | 模块标识符字段。 |
| `version` | *未设置* | [语义版本对象](#版本对象)或字符串 | 模块版本。 |

## 版本对象

`version`对象使用以下字段：

| 字段 | 默认值 | 类型 |
|------|--------|------|
| `buildMeta` | *空* | 字符串 |
| `major` | `0` | 整数 |
| `minor` | `0` | 整数 |
| `patch` | `0` | 整数 |
| `preRelease` | *空* | 字符串 |
