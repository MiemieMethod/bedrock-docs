## 常用标签说明

以下是地物规则和生成规则中最常用的生物群系标签及其用途说明：

/// define
`overworld`

- 所有非变异主世界生物群系。地物规则中最常见的过滤条件之一，用于限定地物仅在主世界中生成。

`overworld_generation`

- 部分变异（`mutated`）主世界生物群系。为确保地物覆盖所有主世界变体，地物规则通常同时检测`overworld`和`overworld_generation`。

`nether`

- 所有下界生物群系。

`monster`

- 大多数允许生物生成的生物群系。

`ocean`

- 所有海洋生物群系（深海与浅海）。

`forest`

- 所有森林类生物群系，包括桦树林、黑森林、针叶林等。

`taiga`

- 所有针叶林类生物群系。

`desert`

- 沙漠类生物群系。

`jungle`

- 丛林类生物群系。

`mesa`

- 恶地类生物群系。

`savanna`

- 热带草原类生物群系。

`swamp`

- 沼泽类生物群系。

`mountains`

- 新版山地类生物群系（包括冰封山峰、草甸等）。

`caves`

- 地下洞穴特殊生物群系（繁茂洞穴、溶洞等）。
