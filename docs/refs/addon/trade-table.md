# 交易表

本页列出行为包`trading/`目录中交易表文件的主要字段。交易表通常由实体组件`minecraft:economy_trade_table`或`minecraft:trade_table`引用，其中官方教程建议新内容优先使用`minecraft:economy_trade_table`。

## 根对象

| 字段 | 类型 | 默认值 | 描述 |
| --- | --- | --- | --- |
| `format_version` | 字符串 | 未设置 | 交易表格式版本。官方示例和原版表中不一定显式写出该字段。 |
| `tiers` | 数组 | 未设置 | 交易层级数组。每个层级在交易实体达到对应交易经验后可用。 |
| `weight` | 整数 | 未设置 | 官方生成参考列出的权重字段。原版资源中的随机权重通常位于单个交易对象中。 |

## 层级

层级对象位于`tiers`数组中。第一层级始终可用；后续层级应按经验需求从低到高排列。当交易系统遇到高于实体当前交易经验的层级时，后续层级不会继续检查。

| 字段 | 类型 | 默认值 | 描述 |
| --- | --- | --- | --- |
| `total_exp_required` | 整数 | `0` | 解锁该层级所需的交易实体总交易经验。常见原版阈值包括`0`、`10`、`70`、`150`和`250`。 |
| `groups` | 数组 | 未设置 | 交易组数组。每个组会从候选交易中随机选择若干项。 |
| `trades` | 数组 | 未设置 | 直接位于该层级下的交易数组，不经过交易组。 |

## 交易组

交易组对象位于`tiers > groups`数组中，用于从一个候选交易集合中随机选择交易。

| 字段 | 类型 | 默认值 | 描述 |
| --- | --- | --- | --- |
| `num_to_select` | 整数 | `1` | 从该组的`trades`数组中随机选择的交易数量。官方参考说明常见值为`1`至`2`。 |
| `trades` | 数组 | 未设置 | 该组的候选交易数组。 |

## 交易

交易对象可位于`tiers > trades`或`tiers > groups > trades`数组中。一个交易描述交易实体接收的物品和给予玩家的物品。

| 字段 | 类型 | 默认值 | 描述 |
| --- | --- | --- | --- |
| `wants` | 数组 | 未设置 | 交易实体希望从玩家处接收的物品列表，通常为`1`至`2`项。 |
| `gives` | 数组 | 未设置 | 交易实体给予玩家的物品列表，通常为`1`项。 |
| `max_uses` | 整数 | `12` | 交易被锁定前可使用的最大次数。实体补货后可重新使用。 |
| `reward_exp` | 布尔值 | `true` | 完成交易时玩家是否获得经验球。 |
| `trader_exp` | 整数 | `1` | 完成交易时交易实体获得的交易经验。 |
| `weight` | 整数 | `1` | 位于同一随机候选集合内时，该交易被选中的权重。 |

## 交易物品

交易物品对象位于`wants`或`gives`数组中。对象也可以通过`choice`表示一组选项。

| 字段 | 类型 | 默认值 | 描述 |
| --- | --- | --- | --- |
| `item` | 字符串 | 未设置 | 物品标识符，例如`minecraft:emerald`。官方参考说明该值可以包含类似`minecraft:coal:0`的数据值后缀。 |
| `quantity` | 整数或对象 | `1` | 物品数量。使用整数时表示固定数量；使用对象时表示随机数量范围。 |
| `price_multiplier` | 数值 | `0.05` | 供需导致价格上升时使用的乘数。较大的值会使价格波动更明显。 |
| `filters` | 对象 | 未设置 | 可选过滤器。过滤器通过时，该物品才会包含在交易中。 |
| `functions` | 数组 | 未设置 | 应用于交易物品的函数数组，可用于附魔、生成特定地图等物品变换。 |
| `choice` | 数组 | 未设置 | 可替代的交易物品数组。交易生成时会从该数组中选择物品。 |

### 随机数量

当`quantity`为对象时，该对象定义随机数量范围。

| 字段 | 类型 | 默认值 | 描述 |
| --- | --- | --- | --- |
| `min` | 整数 | `1` | 该交易物品的最小数量。 |
| `max` | 整数 | `1` | 该交易物品的最大数量。 |

## 示例

```json title="交易表"
{
  "tiers": [
    {
      "total_exp_required": 0,
      "groups": [
        {
          "num_to_select": 1,
          "trades": [
            {
              "wants": [
                {
                  "item": "minecraft:chicken",
                  "quantity": 14,
                  "price_multiplier": 0.05
                }
              ],
              "gives": [
                {
                  "item": "minecraft:emerald",
                  "quantity": 1
                }
              ],
              "trader_exp": 2,
              "max_uses": 16,
              "reward_exp": true
            }
          ]
        }
      ]
    }
  ]
}
```
