# 切换设置架构参考

本文档对应Microsoft Learn的`minecraft:togglesetting`结构。

## 字段

| 字段 | 默认值 | 类型 | 说明 |
|------|--------|------|------|
| `default` | *未设置* | 布尔值 | 默认值。 |
| `name` | *未设置* | 字符串 | 设置标识符。 |
| `text` | *未设置* | 字符串 | 显示文本。 |
| `type` | *未设置* | [类型可选值](#类型可选值) | 类型字段。 |

## 类型可选值

| 值 |
|----|
| `dropdown` |
| `label` |
| `slider` |
| `toggle` |
