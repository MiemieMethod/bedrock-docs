# Follow Caravan

> 文档版本：1.21.0.24

Allows the mob to follow mobs that are in a caravan.

## 架构

```mcschema
follow_caravan:
{
  priority "priority"
  speed_multiplier "speed_multiplier"
  entity_types "entity_types"
  integer "entity_count" : opt
}

```
