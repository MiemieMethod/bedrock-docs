# Transformation

> 文档版本：1.21.0.24

Supports rotation, scaling, and translation

## 架构

```mcschema
transformation:
{
  array "rotation" : opt
  {
    number "0..0" : opt
    number "1..1" : opt
    number "2..2" : opt
  }
  array "scale" : opt
  {
    number "0..0" : opt
    number "1..1" : opt
    number "2..2" : opt
  }
  array "translation" : opt
  {
    number "0..0" : opt
    number "1..1" : opt
    number "2..2" : opt
  }
}

```
