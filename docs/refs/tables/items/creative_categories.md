# 创造分类枚举

> 枚举整理版本：1.19.70.22

本页列出方块描述`menu_category.category`字段和物品描述`menu_category.category`字段所接受的创造模式分类字符串枚举值。

| 接口字符串 | 描述 |
|---|---|
| `construction` | 建筑 |
| `nature` | 自然 |
| `equipment` | 装备 |
| `items` | 物品 |
| `all` | 全部（仅物品以及格式版本低于1.19.40的方块可用） |
| `commands` | 命令（仅物品以及格式版本低于1.19.40的方块可用） |
| `none` | 无 |
| `custom` | 自定义<!-- md:flag china --> |
