    BlockExplodeAfterEventSignal¶

    文档版本：1.21.60.21

    BlockExplodeAfterEventSignal类。用于管理与方块爆炸后事件有关的回调函数。

    方法¶

    subscribe

    订阅方块爆炸后事件并执行给定的回调函数。

    subscribe(callback: (arg: BlockExplodeAfterEvent) => void): (arg: BlockExplodeAfterEvent) => void
    callback：(BlockExplodeAfterEvent) => void
    要执行的回调函数。
    返回值：(BlockExplodeAfterEvent) => void
    订阅此事件的回调函数。
    unsubscribe

    移除给定回调函数对方块爆炸后事件的订阅。

    unsubscribe(callback: (arg: BlockExplodeAfterEvent) => void): void
    callback：(BlockExplodeAfterEvent) => void
    要移除的回调函数。
    返回值：void
