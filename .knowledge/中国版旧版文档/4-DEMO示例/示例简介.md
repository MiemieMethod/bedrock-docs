# 示例简介



## 示例下载

下载[离线文档](https://g79.gdl.netease.com/MCDeveloperDocuments20210616.zip)。

下载离线文档后解压，示例DEMO位于“示例”文件夹。



## DemoMod

该文件夹内主要放置了ModSDK模组开发相关的示例

<span id="TutorialMod"></span>

* TutorialMod

  新手入门必备，展示了如何使用聊天信息发放物品

<span id="AwesomeMod"></span>

* AwesomeMod

  进阶必备，实现了一个简单的射击玩法

<span id="CustomBiomesMod"></span>

* CustomBiomesMod

  展示了[自定义维度](../2-ModSDK模组开发/03-自定义游戏内容/07-自定义维度/1-自定义维度.html)，[群系地貌](../2-ModSDK模组开发/03-自定义游戏内容/07-自定义维度/2-群系地貌.html)，[生物生成](../2-ModSDK模组开发/03-自定义游戏内容/07-自定义维度/3-生物生成.html)与[自定义特征](../2-ModSDK模组开发/03-自定义游戏内容/07-自定义维度/4-自定义特征.html)的基本用法

<span id="CustomBlocksMod"></span>

* CustomBlocksMod

  展示了“ModSDK模组开发 - 自定义游戏内容 - 自定义方块”模块的基本用法

<span id="CustomCropMod"></span>

* CustomCropMod

  展示了[自定义农作物](../2-ModSDK模组开发/03-自定义游戏内容/05-自定义方块/3-特殊方块/3-自定义农作物方块.html)的基本用法

<span id="CustomEffectsMod"></span>

* CustomEffectsMod

  展示了[自定义状态效果](../2-ModSDK模组开发/03-自定义游戏内容/08-自定义状态效果.html)的基本用法

<span id="CustomEntityMod"></span>

* CustomEntityMod

  展示了“ModSDK模组开发 - 自定义游戏内容 - 自定义生物”模块的基本用法

<span id="CustomFurnaceMod"></span>

* CustomFurnaceMod

  请配合[自定义熔炉Demo](../2-ModSDK模组开发/03-自定义游戏内容/05-自定义方块/a-自定义熔炉.html)食用

<span id="CustomItemsMod"></span>

* CustomItemsMod

  展示了“ModSDK模组开发 - 自定义游戏内容 - 自定义物品”模块的基本用法

<span id="customMusicDemo"></span>

* customMusicDemo

  展示了[自定义音乐](../2-ModSDK模组开发/03-自定义游戏内容/09-自定义音乐.html)的基本用法

<span id="CustomRangedWeaponMod"></span>

* CustomRangedWeaponMod

  展示了[自定义远程武器](../2-ModSDK模组开发/03-自定义游戏内容/07%20自定义远程武器.html)的基本用法

<span id="portalGateDemo"></span>

* portalGateDemo

  请配合[自定义传送门DEMO](../2-ModSDK模组开发/03-自定义游戏内容/07-自定义维度/5-自定义传送门.html)食用

<span id="UIDemoMod"></span>

* UIDemoMod

  展示了各种UI控件的基本用法，请配合[UI说明文档](../2-ModSDK模组开发/60-UI/4-UI说明文档.html)食用

<span id="LobbyGoodDemo"></span>

* LobbyGoodDemo

  展示了[联机大厅商品Mod文档](../2-ModSDK模组开发/61-联机大厅开发/6-联机大厅商品Mod文档.html)的基本用法

<span id="CustomMapMod"></span>

* CustomMapMod

  展示了小地图的基本用法

<span id="VirtualWorldDemo"></span>

* VirtualWorldDemo （>=1.22版本）

  展示了虚拟世界的基本用法

* MicroBlocksMod

  展示了[微缩模型](../2-ModSDK模组开发/03-自定义游戏内容/10-微缩方块.html)的基本用法
