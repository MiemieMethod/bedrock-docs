# 简介

​		Apollo是用于开发和部署我的世界PE版网络游戏服务器集群的工具。集成功能插件、负载均衡、集群管理、运营指令等功能于一体。使开发者只需专注于MOD业务逻辑，开发更多玩法组件，为我的世界的创新拓展带来更多可能。

​		我们为开发者准备了MCStudio进行Apollo网络服开发的[视频教程](https://cc.163.com/act/m/daily/iframeplayer/?id=5e7428e16a37ca23faf84bc2)，对应本文档创建第一个网络服的完整流程，可以对照着进行学习。
